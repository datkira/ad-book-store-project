
## Installation

Clone code from the remote repository

```bash
  git clone https://github.com/datkira/ad-book-store-project.git
```
    

Link to the root folder:
```bash
  cd ad-book-store-project
```

Install and run Client:

*Start: Root folder*

```bash
  cd client
```

```bash
  npm install
```

```bash
  npm start
```


Install and run Server:

*Start: Root folder*

```bash
  cd server
```

```bash
  npm install
```

```bash
  npm start
```
